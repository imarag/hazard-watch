# Posts App

A full-stack web application for creating and managing posts with user authentication.

## Features

- User registration and login
- JWT authentication with refresh token flow
- Protected routes
- Create, edit, and delete posts
- Search and filter posts
- Responsive UI
- Modern state management with TanStack Query
- Interactive map to view posts

## Tech Stack

### Frontend

- React
- TypeScript
- React Router
- Material UI
- TanStack Query
- Axios

### Backend

- Node.js
- Express
- JWT Authentication

## Project Structure

- `/frontend` – React client
- `/backend` – API server

## Run Locally

### Frontend

```bash
npm install
npm run dev
```
